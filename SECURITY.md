# Security Policy

Pattern Golf is a static, client-side-only site with no backend, no accounts, and no user
data collection beyond what's stored locally in the player's own browser (`localStorage`).

If you find a security issue (e.g. a way to break out of the sandboxed regex evaluation, or an
XSS vector), please open a GitHub issue with reproduction steps. There is no bug bounty; this is
a hobby project maintained by a single developer, but reports are read and fixed promptly.
